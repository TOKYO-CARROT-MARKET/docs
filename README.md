# 귀국짐 거래 플랫폼 기획서 초안

## 1. 프로젝트 한줄 정의

일본에 거주하는 한국인들이 귀국 전 가전·가구·생활용품을 쉽고 빠르게 거래할 수 있도록 돕는 한국어 중심 지역 기반 플랫폼.

---

## 2. 문제 정의

일본에서 한국인이 귀국할 때 아래와 같은 문제가 반복적으로 발생한다.

### 판매자 입장

* 귀국 직전까지 사용해야 하는 물건이 많아 판매 시점 조절이 어렵다.
* 폐기에는 비용과 시간이 든다.
* 중고매입점은 가격이 너무 낮다.
* 일본 서비스는 언어 장벽이 있다.
* 한인 커뮤니티 거래는 검색성과 신뢰성이 떨어진다.

### 구매자 입장

* 막 일본에 온 사람은 생활용품을 한 번에 구하고 싶다.
* 한인 커뮤니티에 흩어진 매물을 찾기 어렵다.
* 지역, 수령일, 사용기간 등 실제 생활에 중요한 정보가 잘 정리되어 있지 않다.

---

## 3. 목표

### 제품 목표

* 한국인이 일본에서 귀국짐을 쉽게 올리고 쉽게 찾을 수 있게 한다.
* 일반 중고마켓이 아닌, “귀국 일정 기반 거래” 경험을 만든다.
* 지역 기반 직거래를 중심으로 거래 부담을 낮춘다.

### MVP 목표

* 실제 판매 글이 올라오고
* 실제 문의가 발생하고
* 실제 직거래가 1건 이상 성사되는 것

---

## 4. 핵심 타겟

### 1차 타겟: 판매자

* 일본에서 한국으로 귀국 예정인 유학생
* 워킹홀리데이 종료 예정자
* 일본 취업/주재 종료 후 귀국 예정자

### 2차 타겟: 구매자

* 막 일본에 온 유학생
* 워홀/취업으로 자취를 시작한 한국인
* 단기 체류 중 저렴하게 생활용품을 구하려는 사람

---

## 5. 핵심 가치 제안

### 기존 방식의 문제

* 카카오톡/오픈채팅/페북 그룹: 검색 어려움, 정리 안 됨
* 지모티/메루카리: 일본어 장벽, 귀국 일정 특화 아님

### 우리 서비스의 차별점

* 한국어 중심 UX
* 귀국일 기반 판매 관리
* 지역별 탐색 최적화
* 생활 시작/정리 상황에 맞는 묶음 거래 지원

---

## 6. MVP 범위

### 꼭 필요한 기능

1. 회원가입/로그인
2. 매물 등록
3. 매물 목록 조회
4. 지역 필터
5. 카테고리 필터
6. 귀국일 또는 판매 가능일 표시
7. 거래 방식 표시

   * 직거래
   * 직접 수령
   * 배송 가능 여부
8. 판매자 연락 기능

   * 오픈채팅 링크
   * 카카오톡 ID
   * 이메일
9. 판매 완료 처리

### MVP에서 제외해도 되는 기능

* 앱 푸시 알림
* 내부 실시간 채팅
* 결제/에스크로
* 리뷰 시스템
* 다국어 지원
* 추천 알고리즘

## 10. 차별화 기능 후보

### 1) 귀국 타임라인 판매

* 귀국일까지 남은 기간에 따라 정리해서 보여주기
* 예: 바로 수령 가능 / 다음 주 수령 가능 / 귀국 직전 수령 가능

### 2) 입국자용 스타터팩

* 혼자 살기 시작하는 사람이 한 번에 가져갈 수 있는 세트
* 예: 밥솥 + 전자레인지 + 책상 + 의자

### 3) 지역 기반 빠른 거래

* 니시신주쿠, 신오쿠보, 다카다노바바 등 한국인 수요가 있는 지역 중심

## 13. 오픈소스 방향

### 왜 오픈소스로 하는가

* 한국인 개발자/디자이너가 같이 참여 가능
* 서비스의 신뢰성과 투명성을 높일 수 있음
* 사이드 프로젝트에서 커뮤니티 프로젝트로 확장 가능

### 오픈소스 운영에서 필요한 것

* README
* 기획 문서
* 기여 가이드(CONTRIBUTING.md)
* 이슈 템플릿
* 코드 스타일 규칙
* MVP 우선순위 정리

### 주의할 점

* 오픈소스라고 해서 방향이 자동으로 정해지지 않음
* 초기에는 제품 방향을 정할 코어 운영자가 꼭 필요함
